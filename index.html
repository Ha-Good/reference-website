The Section Tag

The section tag is an important element of formatting text in HTML. The section tag defines a section in a document. While the content in a section tag will be related, it might not be used if there is a more specific element to relate content on a page. A section should also typically have a heading tag that allows for it to be defined.

The Div Tag

A div tag is similar to a section tag, the div tag defines a division or section in an HTML document. It is then used to group elements to easily format them in CSS.

The Span Tag

The span tag has no specific meaning. When used with a class or id element, it can used to style text in CSS. It is similar to the div, however the div works better with block elements and the span works better with inline elements

Flexbox

Flexbox is a CSS element which allows for (responsive) content within a container to be auto arranged. Flexbox is one dimensional and moves content into rows or columns, and allows content to fit, expand, or shrink into specific areas.

What is CSS grid?

Css Grid is a method to create responsive grid layouts for web design. The main difference between Grid and flexbox is that grid is better for two dimensional layout, whereas Flexbox is better for one-dimensional. Like Flexbox, grid is defined in the css style sheet of a webpage
